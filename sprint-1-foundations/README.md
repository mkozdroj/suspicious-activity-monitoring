## Sprint 1 Deliverables

Sprint 1 focuses on building the database foundation for the Suspicious Activity Monitor system.

### Scope
- design the E/R model
- create all core database objects
- prepare database utility scripts
- load sample test data
- document setup and schema structure

### Deliverables
- E/R diagram added to the repository
- DDL scripts for all database objects
- shell scripts in `/scripts/db/`
- test data including:
    - 50+ transactions
    - 5+ alert rules
    - 10+ watchlist entries
    - mix of clean and suspicious activity
- `README.md` section with schema explanation and setup instructions

### Expected Outcome
By the end of Sprint 1, the team should have a working, documented database with all core objects stored in source control.
